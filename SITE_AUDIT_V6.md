## Visual Hierarchy Issues

### Problems Identified:

**1. Size Hierarchy Okay, Contrast Lacking**
- Headlines are appropriately sized
- But everything else blends together (same color, weight)
- Need more bold/accent color on key elements

**2. No Visual "Anchor Points"**
Research: ["Users scan in F-pattern, need anchor points"](https://clay.global/blog/web-design-guide/visual-hierarchy-web-design)
- Current: Wall of text with cards
- Missing: Icons, numbers, visual breaks

**3. CTAs Don't Pop**
- Button color (cyan) blends with overall blue theme
- Should have higher contrast

### Recommendations:

**Increase Contrast for Key Elements:**
```css
/* Current */
.btn-primary { background: #00d4ff; } /* cyan */
h2 { color: #FFFFFF; } /* white */

/* Recommended */
.btn-primary {
  background: #FFB81C; /* warm orange - stands out */
  color: #001220; /* dark text */
}
/* Or keep cyan but add more prominent shadow/glow */
```

**Add Visual Anchors:**
- Service cards: Add icon above each title
- Process steps: Add icon or illustration
- Stats: Already good with stat boxes
